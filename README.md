## Getting Started

1. Clone this repository

```
git clone https://github.com/glooobal/smaczny-bot
```

2. Install dependencies

```
pnpm install
```

3. Create .env file and fill it with your credentials

```
  discordToken: ""
  discordClientId: ""

  mongoUri: ""
```

4. Start client

```
pnpm run start
```
